# 🚀 Dynamic Portfolio Website

## 📖 Project Overview

Dynamic Portfolio Website is a full-stack web application that allows users to showcase their professional profile, skills, projects, and certifications through a modern and responsive portfolio website.

The project features a secure **Portfolio Manager Login** and a dedicated **Portfolio Editor**, enabling real-time content management without modifying the source code. All portfolio information is stored and managed using Supabase Database, Authentication, and Storage services.

## 🔄 Workflow

1. User visits the portfolio website.
2. Portfolio data is fetched dynamically from Supabase.
3. Authorized users log in through Portfolio Manager Login.
4. Portfolio content is managed using Portfolio Editor.
5. Data is stored in Supabase Database and Storage.
6. Changes are reflected instantly on the portfolio website.
